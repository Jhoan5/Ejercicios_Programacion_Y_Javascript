# Ejercicios_Programacion_Y_Javascript

## Ejercicios de [Carlos](https://github.com/carlosandresalzate/ejercicios_de_Javascript)  

1. Función que muestre la posición, el valor y el indice de un string o array.

2. Función que busca una letra en una palabra, dados un parametro palabra y letra a buscar.

3. Función que retorna string con su indice. Algo como al pasar hoy muestra 'h0o1y2'.

4. Función que une los elementos de un array y retorna un string.

5. Función que reciba array y retorne otro con los tipos de valores en el array.

6. Retornar los valores de un objeto en forma de elemntos en un array.

7. Retornar los valores de un objeto en forma de un string.

8. Realiza un programa que determine la cantidad de vocales que contiene un array de strings.

9. Realiza un programa que reverse los valores de un array o string.

10. Crea una función para la cual la vas a hacer en dos partes.
  * Crea una función que tome un objeto y un array dónde el array filtra las propiedades del objetos y retorna los key encontrados en el objeto del array filtro.
  * A la función anterior añade un parametro de inclución dónde se le diga si debe retornar las keys que hay en el objeto o las que no hay.

11. Función que recibe dos array y devuelve uno solo con los elementos de ambos, pero reversado.

12. Eres contratado por una empresa para desarrollar un algoritmo el cual permita clasificar a sus usuarios. Cada usuario es representado como un objeto de javascript y sus keys son la propiedad del mismo. Por ejemplo:  
  
{  
    nombre: "Homero Simpson",  
    email: "homer@fox.com",  
    edad: 33,  
    direccion: "Siempreviva 123"  
}  
  
Se requiere escribir una función filtrarUsuarios que reciba un array arr como primer parámetro y un objeto filtro como segundo parámetro. El array arr tendrá la siguiente forma:\  

[usuario1, usuario2, usuario3, ...]  

y el objeto filtro:  

{  
    edad: 33  
}  
  
La función filtrarUsuarios deberá retornar un array (uno nuevo) que contenga a todos los usuarios que coincidan con lo valores que posee el objeto filtro.  

13. Una función que recibe un array y que retorna los elementos de ese array cuyo indice sea par.
