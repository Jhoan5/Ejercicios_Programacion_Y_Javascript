## Otros ejercicios de lógica de programación

1. Crea las siguiente funciones, estas se relacionan un poco con matematicas.
  1.  Una función que diga si un numero es par o impar.
  2. Una función que retorne cuales son los numeros pares o impares de un array.
  3. Una función que determine los divisores de un número.
  4. Una función que imprima la tabla de multiplicar de un número.
  5. Una calculadora que pueda sumar, restar, multiplicar y dividir dos números.
